### 🚗 Featured Project

#### [데이터 기반 중고차 구매 의사결정 지원 서비스](https://github.com/SKN26-1st-1team/used_car)

> 단순 매물 조회가 아닌, **"이 가격이 합리적인가?"** 를 해석해주는 서비스

- 🕸️ 보배드림 크롤링 → MySQL 적재 → Streamlit 서비스까지 end-to-end 구현
- 📐 연식/주행거리를 **감가 단위로 환산한 유사도 거리**(`|Δyear| + |Δkm|/22,000`)로 비교군 구성
- 📊 군집 내 **기대가격 회귀 + 분위수(Q1/Q3) 판정**으로 저평가·적정·고평가 해석 제공
- 🧩 선행연구(Kihm & Vance, 2016) 기반 룰 계수 채택 — 표본 제약 환경에서의 **설계 판단**
